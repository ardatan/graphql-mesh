This page is shown be default whenever a 404 is hit.
You can disable this by behavior via the landingPage option. 

          
import { defineConfig as defineServeConfig } from '@graphql-mesh/serve-cli';

export const serveConfig = defineServeConfig({
  landingPage: false
})

If you expected this page to be the GraphQL route, you need to configure Mesh. Currently, the GraphQL route is configured to be on __GRAPHIQL_LINK__. 

          
import { defineConfig as defineServeConfig } from '@graphql-mesh/serve-cli';

export const serveConfig = defineServeConfig({
  graphqlEndpoint: '__REQUEST_PATH__',
})
