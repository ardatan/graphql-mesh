# Deduplicate HTTP Requests

This plugin deduplicates the same HTTP requests done in parallel to reduce the network traffic.

## Getting Started

```sh npm2yarn
npm i @graphql-mesh/plugin-deduplicate-request
```

```yaml filename=".meshrc.yaml"
plugins:
  - deduplicateRequest: {}
```
