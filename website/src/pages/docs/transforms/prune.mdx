# Prune Transform

The `prune` transform allows you removing unused and empty types.

```sh npm2yarn
npm i @graphql-mesh/transform-prune
```

## How to use?

Add the following configuration to your Mesh config file:

```yaml filename=".meshrc.yaml" {2-3}
transforms:
  - prune:
      skipPruning: []
```
