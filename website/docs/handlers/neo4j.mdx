This handler allows you to use the GraphQL schema created by [`@neo4j/graphql`](https://github.com/neo4j/graphql).

To get started, install the handler library:

<PackageInstall packages="@graphql-mesh/neo4j" />

Now, you can use it directly in your Mesh config file:

```yaml
sources:
  - name: Neo4j
    handler:
      neo4j:
        url: neo4j://localhost
        username: neo4j
        password: MY_PASSWORD
```
